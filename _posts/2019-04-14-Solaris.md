# SunOS 5.10

## SSH连接过慢解决办法

编辑：/etc/ssh/sshd_config

OpenSSH: `UseDNS no`

Solaris: `LookupClientHostnames no`

Solaris重启sshd服务：

```bash
/lib/svc/method/sshd { start | restart } 
```

## 去除文件中所有^M

```bash
perl -pi -e 's/^M//g' filename
```

## 批量去除^M符号

```bash
find . -exec perl -pi -e 's/^M//g' '{}' \;
```

**注意：^M的输入方式是 Ctrl + v ，然后Ctrl + m**
