# 快速安装

## windows

在官网直接下载msi文件安装即可：https://golang.org/dl/

## ubuntu

如果您使用的是Ubuntu 16.04 LTS，18.04 LTS或19.04，那么您可以使用longsleep/golang-backportsPPA并安装Go 1.13。

    add-apt-repository ppa:longsleep/golang-backports
    apt-get update
    apt-get install golang-go

# 国内不能go get处理办法
