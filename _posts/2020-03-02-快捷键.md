# jetbrains

如果你要查找文件,可以使用快捷键: ctrl+shift+n, 输入文件名称,可以快速搜索文件;

也可以Ctrl+e,打开最近编辑文件,打开之后你就能知道具体路径了,

Ctrl+左键 点击文件,会出现一个file path 这样能直接打开文件夹



# VScode

复制一行    | ctrl+c   (光标未选状态下)

格式化代码  | Shift + Alt + F

行选择      | win 下 alt + shift +鼠标拖放 /点选

全局查找    | Shift + Ctrl + F

多行注释    | Shift + Alt + A
